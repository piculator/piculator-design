# 安装中文支持软件包

## 中文字体

```bash
sudo apt -y install fonts-wqy-zenhei fonts-wqy-microhei
sudo fc-cache
```

## 输入法

```bash
sudo apt-get install fcitx fcitx-googlepinyin
im-config
```
